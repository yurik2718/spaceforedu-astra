# Images — SpaceForEdu

All images must be **CC0 / free for commercial use** with no attribution required.
Approved sources: Unsplash, Pexels, Pixabay.

## Folder structure

```
images/
├── hero/               Hero section — main landing page
├── services/
│   ├── homologacion/   Homologación de títulos service
│   ├── universidad/    University admission service
│   └── espanol/        Spanish language courses
├── lifestyle/          Spain lifestyle, cities, student life
├── team/               Team / consultant portraits (if real photos available)
├── og/                 Open Graph images (1200×630 px)
├── flags/              Country flag SVGs (existing)
└── universities/       University logo SVGs/PNGs (existing)
```

## Naming convention

`{section}-{descriptor}-{size}.webp`

Examples:
- `hero/hero-students-madrid-1920.webp`
- `services/homologacion/docs-ministry-800.webp`
- `lifestyle/barcelona-campus-800.webp`

## Required sizes

| Use case      | Width   | Format |
|---------------|---------|--------|
| Hero (desktop)| 1920 px | .webp  |
| Hero (mobile) | 768 px  | .webp  |
| Service card  | 800 px  | .webp  |
| OG image      | 1200 px | .jpg   |

Convert all originals to .webp before committing (use Squoosh or `cwebp`).
